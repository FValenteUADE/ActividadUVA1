Mi Portafolio

Este primer proyecto trata del programa "Censo 2022" (Censo Argentina 2022).

El mismo fue creado utilizando Java 8 y un patrón de diseño "MVC" (Model, View, Controller) para el uso de un "controlador" que mediara entre las clases del "Frontend" y las del "Backend".

El programa consiste en asignar manzanas de un radio censal a la menor cantidad de censistas. Para lo cual en principio debe importarse un radio censal, el que luego podrá verse en el mapa.

Para obtener una respuesta exacta y que asegure ser la minima cantidad de censistas requerias sería necesario obtener un recorrido óptimo en la asignación (para que siempre cada censista reciba tres manzanas de radio). Para esto aplicamos un criterio goloso que relega la decisión de cuáles serán las próximas manzanas.
