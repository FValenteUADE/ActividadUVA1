Diseño Completo de un Sitio

Teniendo ya experiencia como desarrollador en distintos rubros ofrezco crear un sitio web de manera completa.

Esto incluye:

  • Desarrollo de Frontend (JavaScript, CSS, HTML utilizando de framework "Angular").
  • Desarrolo de Backend (Python, Java, Spring y SpringBoot).
  • CI/CD (Jenkins y SonarQube).
  • Despliegue en una nube (AWS).
  • Diseño y creación de una Base de Datos Relacional (Oracle) o No Relacional (MongoDB).
  • Versionado del aplicativo (Git con GitHub o BitBucket).

Diseño Completo de una Base de Datos

Teniendo ya experiencia diseñando y creando bases de datos puedo darte un diseño completo incluyendo:

  • Normalización de la DB.
  • Atención a los tiempos de respuesta del Backend a través de la creación de Índices y Particiones.
  • Validación del diseño a través de PowerDesigner (PWD).
  • Despliegue en una nube (AWS).
